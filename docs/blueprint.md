# **App Name**: BudgetWise

## Core Features:

- Transaction Input: Allow users to manually input income and expenses, specifying amount, category, and date.
- Budget Creation: Enable users to create monthly budgets for different spending categories (e.g., groceries, rent, entertainment).
- Spending Tracker: Automatically track spending against the defined budget for each category.
- AI-Powered Spending Suggestions: The app can offer users targeted advice and options that act as a tool in helping them with strategies to cut their spending based on historical data, like reducing discretionary spending by a certain percentage or finding better deals on recurring bills.
- Report Generation: Generate reports showing income, expenses, and budget status over time, with graphical visualizations.
- Alerts & Notifications: Send notifications for overspending in categories and upcoming bill payments.

## Style Guidelines:

- Primary color: Forest green (#38A3A5) to inspire trust and financial security.
- Background color: Light off-white (#F0F7F4) for a clean, unobtrusive feel.
- Accent color: Muted gold (#D9B043) for highlighting key financial data and CTAs.
- Body and headline font: 'PT Sans' for readability and a modern feel.
- Use flat, line-based icons for categories and actions.
- A clean, tab-based layout separating transaction input, budget overview, and reports.
- Smooth transitions when navigating between sections and updating charts.
